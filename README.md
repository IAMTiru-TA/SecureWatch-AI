# 🚀 SecureWatch AI

An AI-powered cybersecurity news monitoring system that automatically collects cybersecurity news, analyzes it using AI, and displays structured security alerts through a modern web dashboard.

---

## 📌 Overview

SecureWatch AI automates the entire cybersecurity news monitoring process.

Instead of manually reading dozens of security blogs every day, the workflow automatically:

1. Collects cybersecurity news
2. Extracts article information
3. Sends the content to an AI model for analysis
4. Generates structured security alerts
5. Displays everything on a modern dashboard

This project demonstrates AI workflow automation using **n8n**, **Groq AI**, REST APIs, and a **Next.js** frontend.

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/yourusername/securewatch-ai.git
```

### Navigate to the project

```bash
cd securewatch-ai
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```
